# Haoyu Yang — Personal Website

A single-page academic site. To publish on GitHub Pages:

1. Create a repo named `haoyuyang.github.io` (or any repo).
2. Upload `index.html` and the `assets/` folder, keeping the structure.
3. In the repo: **Settings → Pages → Source: Deploy from branch → main / root.**
4. Your site goes live at `https://<username>.github.io/`.

To update content (publications, talks, etc.), edit `index.html` — the
publication list lives in the `pubs` array near the bottom of the file.
